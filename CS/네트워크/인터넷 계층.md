### ICMP

- Internet Control Message Protocol
- 노드사에이서 통신이 잘 되는지 확인할 때 쓰는 프로토콜
- 데이터를 교환하는데 사용되지 않는 프로토콜 -> 테스팅에 사용
- IP 와는 달리, TCP 또는 UDP와 같은 [전송 계층](전송%20계층.md) 프로토콜과 연관되지 않고, 독립적인 비연결형 프로토콜
	- 대표적으로 ping은 ICMP로 통신
